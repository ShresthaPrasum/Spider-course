Trigonometric Ratios & Identities

1. Basic Ratios
sin(θ) = P / H cosec(θ) = H / P
cos(θ) = B / H sec(θ) = H / B
tan(θ) = P / B cot(θ) = B / P
2. Reciprocal Identities
sin(θ) = 1 / cosec(θ) cosec(θ) = 1 / sin(θ)
cos(θ) = 1 / sec(θ) sec(θ) = 1 / cos(θ)
tan(θ) = 1 / cot(θ) cot(θ) = 1 / tan(θ)
sin(θ) ⋅ cosec(θ) = 1 cos(θ) ⋅ sec(θ) = 1
tan(θ) ⋅ cot(θ) = 1 tan(θ) = sec(θ) / cosec(θ)
3. Quotient Identities
tan(θ) = sin(θ) / cos(θ) cot(θ) = cos(θ) / sin(θ)
sin(θ) = tan(θ) ⋅ cos(θ) cos(θ) = sin(θ) ⋅ cot(θ)
tan2(θ) = sin2(θ) / cos2(θ) cot2(θ) = cos2(θ) / sin2(θ)
sin2(θ) = tan2(θ) / (1 + tan2(θ)) cos2(θ) = 1 / (1 + tan2(θ))
4. Pythagorean Identities
sin2(θ) + cos2(θ) = 1 sin2(θ) = 1 - cos2(θ)
tan2(θ) + 1 = sec2(θ) sec2(θ) - tan2(θ) = 1
cot2(θ) + 1 = cosec2(θ) cosec2(θ) - cot2(θ) = 1
cos(θ) = √(1 - sin2(θ)) sin(θ) = √(1 - cos2(θ))
tan2(θ) = (1 - cos2(θ)) / cos2(θ) cot2(θ) = (1 - sin2(θ)) / sin2(θ)
